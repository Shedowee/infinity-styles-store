Infinity Styles

Infinity Styles is a modern, interactive online store built with React. It features a seamless shopping experience allowing users to browse clothes and electronics, view detailed product information, and manage the store inventory via a dedicated Admin Panel.

For Users

Interactive Modals: View product details without leaving the page.

Toast Notifications: Get instant feedback for actions (e.g., adding to cart, errors).

Responsive Design: Optimized for a great experience on desktop, tablet, and mobile.

For Admins

Product Management: Add, edit, and delete products from the inventory.

User Management: View and manage registered users.

Dashboard: Centralized control over store data.

ğŸ“¦ Installation & Setup

Follow these steps to get the project up and running locally.

Clone the repository

git clone [https://github.com/YourUsername/infinity-styles.git](https://github.com/YourUsername/infinity-styles.git)
cd infinity-styles


Install Dependencies

npm install


Start the Backend (JSON Server)
Open a terminal and run the mock server (runs on port 3001 or 5000 typically, check package.json):

npm run server


Note: Ensure db.json exists in the root or data folder.

Start the React App
Open a second terminal window and run:

npm start


View the App
Open http://localhost:3000 to view it in the browser.
